## Pagina-Web-Gaming
Trabajo hecho para el ramo Desarrollo Web + IA en cual consiste en la solucion de problemas tanto funcionales como visuales en un codigo HTML convinando CSS.

> [!NOTE] use un . para evitar poner el simbolo "mayor que" por que github me lo marcaba como texto especial.

Se realizaron varias mejoras tanto de codigo como visuales como el apartado de listas, en el codigo original las listas estaban organizadas de una manera desfavorable para el plan visual que queria como resultado, por lo cual tuve que reorganizarlas un poco (Recuerdo que las <li estaban asi *"<li class="empresas".<h2.Nintendo</h2.</li. resto codigo..."* , esto me causaba problemas de organizacion asi que para mejorarlo **añadi la imagen, parrafo y header dentro del <li.** para que asi tuviera "bloques" que se pudieran apilar horizontalmente, ese fue el problema principal que mas tarde en darme cuenta.

De resto aparte de lo que se pedia en la actividad *(como integrar los colores de texto desde el archivo CSS)* hice algunas clases para mejorar el apartado visual, tardando en adaptarme al principio pero fluyendo como nunca durante el resto del proceso, de hecho revisando el CSS se puede notar un poco de evoluvion creativa de mi parte al intentar centrar una imagen para que al achicar la ventana no quedara en el mismo lugar y la imagen se moviera con este con *"margin-left: 33% & margin-right:33%"* que luego evolucione a crear listas horizontales como *"ul {display: flex; ... justify-content: space-around;}"* para mas automatizacion.

Para ayuda use el material de estudio dado en clases, Google y Youtube, evite lo maximo para usar IA.
